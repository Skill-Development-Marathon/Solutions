# [2221. Find Triangular Sum of an Array](https://leetcode.com/problems/find-triangular-sum-of-an-array/)

#### Ruby

def triangular_sum(nums)
  new_array = sum_all_numbers(nums)
  if new_array.size == 1
      return new_array.last
  else
      triangular_sum(new_array)
  end
end

def sum_all_numbers(array)
  new_array = Array.new
  (array.size - 1).times do |i|
      new_array << (array[i] + array[i + 1]) % 10
  end
  array.size > 1 ? new_array : array
end
